API Ulasan Film

Selamat datang di dokumentasi API ulasan film. API ini memungkinkan Anda untuk mengelola ulasan film.

Endpoints

GET /

Menampilkan halaman dokumentasi ini.

GET /reviews

Mendapatkan semua ulasan film.

Contoh Respon:


[
    {
        "id": 1,
        "film_id": "86e544fd-79de-4e04-be62-5be67d8dd92e",
        "user": "Andi",
        "rating": 5,
        "comment": "Film animasi terbaik sepanjang masa!"
    }
]

GET /reviews/:id

Mendapatkan ulasan film berdasarkan ID.

Parameter:

  • id (integer): ID unik dari ulasan.

Contoh Respon:


{
    "id": 1,
    "film_id": "86e544fd-79de-4e04-be62-5be67d8dd92e",
    "user": "Andi",
    "rating": 5,
    "comment": "Film animasi terbaik sepanjang masa!"
}

POST /reviews

Membuat ulasan baru.

Body Request:


{
    "film_id": "string",
    "user": "string",
    "rating": "integer",
    "comment": "string"
}

Contoh Respon:


{
    "id": 2,
    "film_id": "...",
    "user": "...",
    "rating": "...",
    "comment": "..."
}

PUT /reviews/:id

Memperbarui ulasan yang sudah ada.

Parameter:

  • id (integer): ID unik dari ulasan.

Body Request:


{
    "film_id": "string",
    "user": "string",
    "rating": "integer",
    "comment": "string"
}

DELETE /reviews/:id

Menghapus ulasan.

Parameter:

  • id (integer): ID unik dari ulasan.
